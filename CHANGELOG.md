## [1.0.0] - {PR_MERGE_DATE}

Initial release ported from [utools-raindrop](https://github.com/jae-jae/utools-raindrop).

### Added

- **Search Bookmarks** — Search Raindrop.io bookmarks with instant local fuzzy matching and API-powered results
- **Browse Collections** — Navigate collections, sub-collections, and view bookmark details with metadata panel
- **Add Bookmark** — Save new bookmarks with auto-fill from clipboard or selected text
- Local MRU cache for recently accessed bookmarks (max 100 items)
- Results sorted by most recently used first
- Token-based authentication via Raycast Preferences
